# Icon Source Files

このディレクトリには、Tools アプリのアイコンのソースファイルが含まれています。

## ファイル構成

- `*.xcf` - GIMP 形式のソースファイル
- `*.png` - 中間出力ファイル（参照用）

## 使用方法

これらのソースファイルから最終的なアイコンを生成し、`public/` ディレクトリに配置します。

### 生成されるファイル

- `public/favicon.ico` - ブラウザのファビコン
- `public/icon-192x192.png` - PWA用アイコン (192x192)
- `public/icon-512x512.png` - PWA用アイコン (512x512)

## 編集方法

1. GIMP で .xcf ファイルを開く
2. 編集を行う
3. 各サイズにエクスポート
4. 最終ファイルを `public/` ディレクトリに配置
