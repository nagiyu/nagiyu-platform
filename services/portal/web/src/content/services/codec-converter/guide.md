## Codec Converter の使い方ガイド

このページでは、Codec Converter を使って動画・音声ファイルを変換する手順を解説します。

## 事前準備

- nagiyu アカウントでのログインが必要です。
- 変換したいファイルを手元に準備してください。
- 出力したいコーデック・フォーマットを事前に決めておくとスムーズです。

---

## 動画変換の手順

### ステップ 1: ダッシュボードへのアクセス

[https://codec-converter.nagiyu.com](https://codec-converter.nagiyu.com) にアクセスしてログインします。

### ステップ 2: ファイルのアップロード

1. 「新しい変換を開始」ボタンをクリックします。
2. 「動画変換」タブを選択します。
3. ファイル選択ダイアログから変換したい動画ファイルを選択するか、ドラッグ＆ドロップします。
4. 対応フォーマット: MP4・MOV・AVI・MKV・WebM・FLV
5. 最大ファイルサイズ: 10 GB

### ステップ 3: 出力コーデックの選択

変換後の動画コーデックを選択します。

| コーデック | 特徴               | 推奨用途           |
| ---------- | ------------------ | ------------------ |
| H.264      | 高互換性・標準品質 | 一般的な動画共有   |
| H.265      | 高圧縮・高画質     | 4K動画・保存用     |
| VP9        | オープン・高品質   | Web配信・YouTube   |
| AV1        | 最高圧縮率         | ストリーミング配信 |

### ステップ 4: 品質設定

- **解像度**: 元の解像度を維持 / 4K / 1080p / 720p / 480p / 360p
- **ビットレート**: 自動（推奨）/ カスタム（kbps 単位で指定）
- **フレームレート**: 元のフレームレートを維持 / 60fps / 30fps / 24fps
- **CRF（品質係数）**: 0（最高品質）〜 51（最低品質）。デフォルト: 23

### ステップ 5: 音声設定

- **音声コーデック**: AAC（推奨）/ MP3 / Opus / FLAC / 音声なし
- **音声ビットレート**: 320kbps / 256kbps / 192kbps / 128kbps
- **チャンネル数**: ステレオ / モノラル

### ステップ 6: 変換の実行

「変換を開始する」ボタンをクリックします。AWS Batch ジョブが開始され、処理状況がリアルタイムで確認できます。

### ステップ 7: ダウンロード

処理完了後、「ダウンロード」ボタンから変換済みファイルを保存します。

## 音声変換の手順

### ステップ 1〜2: アップロードまでは同様

「音声変換」タブを選択してファイルをアップロードします。対応フォーマット: MP3・AAC・FLAC・WAV・OGG・M4A・AIFF

### ステップ 3: 出力フォーマットの選択

| フォーマット | 特徴               |
| ------------ | ------------------ |
| MP3          | 最高互換性         |
| AAC          | 高効率・Apple標準  |
| Opus         | 低遅延・高品質     |
| FLAC         | 可逆圧縮（無劣化） |

### ステップ 4: 品質設定

- **サンプリングレート**: 44.1kHz / 48kHz / 96kHz
- **ビットレート**: 320kbps / 256kbps / 192kbps / 128kbps / 64kbps

---

## 変換状況の確認・管理

ダッシュボードの「変換履歴」から過去のジョブ一覧と状態を確認できます。失敗したジョブは「再試行」ボタンで再実行できます。変換済みファイルの保存期間は生成から 7 日間です。
