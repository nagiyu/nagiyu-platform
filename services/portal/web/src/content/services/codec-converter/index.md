## Codec Converter とは

Codec Converter は、動画・音声ファイルをさまざまなコーデックやフォーマットに変換できるオンラインコンバーターサービスです。専用ソフトをインストールすることなく、ブラウザからファイルをアップロードするだけで変換処理を開始できます。変換処理は AWS Batch を活用したサーバーレスアーキテクチャで実行されるため、CPU 負荷の高いコーデック変換もブラウザ側に負担をかけず処理できます。

## 対応コーデック・フォーマット

### 動画コーデック

- **H.264 (AVC)**: 最も広く普及した動画コーデック。高い互換性を持ち、Web・モバイル・放送など幅広い用途に対応。
- **H.265 (HEVC)**: H.264 の後継。同等画質でファイルサイズを約 50% 削減。4K 動画に適している。
- **VP9**: Google が開発したオープンコーデック。YouTube での採用実績が豊富。
- **AV1**: 次世代オープンコーデック。高圧縮率と高画質を両立し、ストリーミング配信に最適。

### 音声コーデック

- **MP3**: 最も普及している音声フォーマット。互換性が高く、一般的な音楽再生に適している。
- **AAC**: MP3 より高効率な音声コーデック。iTunes・Apple Music などで標準採用。
- **Opus**: 低遅延・高品質な音声コーデック。WebRTC や Discord などでの採用実績あり。
- **FLAC**: 可逆圧縮音声フォーマット。音質を損なわずにファイルサイズを削減。

### コンテナフォーマット

MP4・WebM・MKV・MOV・AVI・MP3・AAC・FLAC・OGG に対応。

## 主な変換ユースケース

### ストリーミング配信向け最適化

H.264 で録画した動画を AV1 に変換することで、同等画質でファイルサイズを大幅に削減し、配信コストを抑えられます。

### 古い形式から新しい形式への変換

AVI・MOV など古いフォーマットの動画を、現在の Web 標準である MP4（H.264）や WebM（VP9）に変換できます。

### 音声抽出

動画ファイルから音声トラックのみを抽出して MP3・AAC・FLAC として保存できます。

## サービスの仕組み

1. **アップロード**: 変換したいファイルを Amazon S3 にアップロードします。
2. **変換設定**: 出力コーデック・品質・解像度などを設定します。
3. **処理実行**: AWS Batch が自動起動し、FFmpeg で変換処理を行います。
4. **完了通知**: 変換完了後にダッシュボードと通知でお知らせします。
5. **ダウンロード**: 変換済みファイルをダウンロードします。

- 動画クリエイター・配信者
- Web 開発者（動画アセットの最適化）
- 映像制作・編集者
- 音楽制作者・ポッドキャスター

## 背景・開発動機

動画のコーデック変換は、Web 開発や配信用の素材準備などで頻繁に必要になる作業です。私自身が普段の開発や配信準備の中で「MP4 を AV1 に変換したい」「古い形式の動画を現在の Web 標準に合わせたい」という場面に繰り返し遭遇していました。こうした変換には従来 FFmpeg を手元にインストールして CLI コマンドを組み立てる必要があり、専門知識がない人にとってはハードルが高い作業です。専用のデスクトップアプリを使う方法もありますが、インストールやライセンス管理の手間がかかります。そこで、ブラウザだけで手軽に変換が完結し、かつ重い処理もサーバー側で引き受けてくれるサービスを自分で作ることにしました。変換処理を AWS Batch に委ねることで、ブラウザや端末への負荷なく大きなファイルも扱えるようにしています。

## 想定ユーザー

私がこのサービスを作る際に想定したのは、動画変換の必要性はあるものの FFmpeg などの専門ツールを日常的に使い慣れていない人たちです。具体的には、配信用の動画を AV1 や VP9 に変換してファイルサイズを抑えたい動画クリエイター、Web サイトや Web アプリ向けに動画アセットを最適化したい開発者、撮影・編集した素材を特定のフォーマットに変換して納品したい映像制作者などを想定しています。アカウント登録なしで使い始められるので、「とりあえず 1 本だけ変換したい」というライトな使い方にも対応できます。また、変換処理がブラウザ上ではなくサーバー側の AWS Batch で実行されるため、処理中にブラウザを閉じても問題なく、作業の合間に変換をかけておくといった使い方も自然にできます。

## 他ツールとの違い

Codec Converter の特徴は、重い変換処理をすべてサーバー側（AWS Batch）で実行する点にあります。ブラウザ上での変換処理は端末の CPU を占有し、処理中はブラウザが固まることがありますが、本サービスでは変換を開始した後はブラウザを閉じてもサーバー側で処理が継続します。また、アカウント登録が不要でブラウザだけで使い始められるため、ソフトウェアのインストールが不要です。入力に MP4 のみ対応（最大 500MB）というシンプルな構成の一方で、出力コーデックは H.264・VP9・AV1 と主要な動画コーデックをカバーしており、ストリーミング配信向けの最適化からアーカイブ用の高品質変換まで幅広く対応できます。
