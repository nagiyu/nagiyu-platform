## Codec Converter とは

Codec Converter は、MP4 動画ファイルを H.264・VP9・AV1 などのコーデックに変換できるオンラインコンバーターサービスです。専用ソフトをインストールすることなく、ブラウザからファイルをアップロードするだけで変換処理を開始できます。変換処理は AWS Batch を活用したサーバーレスアーキテクチャで実行されるため、CPU 負荷の高いコーデック変換もブラウザ側に負担をかけず処理できます。

## 対応コーデック・フォーマット

### 入力フォーマット

- **MP4**: 入力ファイルは MP4 のみに対応しています。最大ファイルサイズは 500MB です。

### 出力コーデック

- **H.264 (AVC)**: 最も広く普及した動画コーデック。高い互換性を持ち、Web・モバイル・放送など幅広い用途に対応。出力コンテナ: .mp4
- **VP9**: Google が開発したオープンコーデック。YouTube での採用実績が豊富。出力コンテナ: .webm
- **AV1**: 次世代オープンコーデック。高圧縮率と高画質を両立し、ストリーミング配信に最適。出力コンテナ: .webm

## 主な変換ユースケース

### ストリーミング配信向け最適化

H.264 で録画した動画を AV1 に変換することで、同等画質でファイルサイズを大幅に削減し、配信コストを抑えられます。

### Web 標準コーデックへの変換

H.264 で録画・保存された MP4 動画を、現在の Web 標準である VP9 や AV1（WebM）に変換できます。

## サービスの仕組み

1. **アップロード**: 変換したいファイルを Amazon S3 にアップロードします。
2. **コーデック選択**: ラジオボタンで出力コーデック（H.264 / VP9 / AV1）を選択します。品質パラメータ（CRF 値）はコーデックごとに固定です。
3. **処理実行**: AWS Batch が自動起動し、FFmpeg で変換処理を行います。
4. **ダウンロード**: 処理完了後、ジョブ詳細ページから変換済みファイルをダウンロードします。

## 対象ユーザー

- 動画クリエイター・配信者
- Web 開発者（動画アセットの最適化）
- 映像制作・編集者

## 背景・開発動機

動画のコーデック変換は、Web 開発や配信用の素材準備などで頻繁に必要になる作業です。私自身が普段の開発や配信準備の中で「MP4 を AV1 に変換したい」「古い形式の動画を現在の Web 標準に合わせたい」という場面に繰り返し遭遇していました。こうした変換には従来 FFmpeg を手元にインストールして CLI コマンドを組み立てる必要があり、専門知識がない人にとってはハードルが高い作業です。専用のデスクトップアプリを使う方法もありますが、インストールやライセンス管理の手間がかかります。そこで、ブラウザだけで手軽に変換が完結し、かつ重い処理もサーバー側で引き受けてくれるサービスを自分で作ることにしました。変換処理を AWS Batch に委ねることで、ブラウザや端末への負荷なく大きなファイルも扱えるようにしています。

## 想定ユーザー

私がこのサービスを作る際に想定したのは、動画変換の必要性はあるものの FFmpeg などの専門ツールを使い慣れていない人たちです。配信用の動画を AV1 や VP9 に変換したい動画クリエイター、Web アプリ向けに動画アセットを最適化したい開発者、素材を特定フォーマットに変換して納品したい映像制作者を主なターゲットにしています。アカウント登録なしで使い始められるので「とりあえず 1 本だけ変換したい」というライトな用途にも対応できます。変換処理はサーバー側（AWS Batch）で実行されるため、処理中にブラウザを閉じても問題なく、作業の合間に変換をかけておくといった使い方も自然にできます。

## 他ツールとの違い

Codec Converter の特徴は、重い変換処理をすべてサーバー側（AWS Batch）で実行する点にあります。ブラウザ上での変換処理は端末の CPU を占有し、処理中はブラウザが固まることがありますが、本サービスでは変換を開始した後はブラウザを閉じてもサーバー側で処理が継続します。また、アカウント登録が不要でブラウザだけで使い始められるため、ソフトウェアのインストールが不要です。入力に MP4 のみ対応（最大 500MB）というシンプルな構成の一方で、出力コーデックは H.264・VP9・AV1 と主要な動画コーデックをカバーしており、ストリーミング配信向けの最適化からアーカイブ用の高品質変換まで幅広く対応できます。
