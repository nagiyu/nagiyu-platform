## nagiyu Admin とは

nagiyu Admin は、nagiyu プラットフォーム全体を管理するための管理画面サービスです。プラットフォーム管理者がサービス設定・ユーザー管理・システム監視などを行うための総合ダッシュボードを提供します。[https://admin.nagiyu.com](https://admin.nagiyu.com) からアクセスできます（管理者権限が必要です）。

## 主な機能

### ユーザー管理
nagiyu プラットフォームに登録しているユーザーの一覧表示・詳細確認・権限管理を行います。

- **ユーザー一覧**: 全ユーザーの登録情報・最終ログイン日時・利用サービスを確認
- **権限管理**: ユーザーに管理者権限を付与・剥奪する
- **アカウント停止**: 不正利用ユーザーのアカウントを一時停止・削除する
- **利用統計**: ユーザーごとの各サービス利用状況を確認

### サービス設定
各 nagiyu サービスの設定を管理画面から変更します。

- **機能フラグ**: 特定の機能をオン・オフ切り替える（フィーチャーフラグ管理）
- **制限設定**: ファイルサイズ上限・処理時間上限・1日の処理件数上限などを設定
- **メンテナンスモード**: 各サービスのメンテナンスモードを切り替える

### システム監視
プラットフォーム全体の健全性をリアルタイムで監視します。

- **サービス稼働状況**: 各サービスの稼働状態（正常・警告・停止）を一覧表示
- **AWS リソース監視**: EC2・ECS・Lambda・RDS などの稼働状況と使用率を確認
- **エラーログ閲覧**: 各サービスのエラーログをリアルタイムで確認
- **アラート設定**: 異常検知時にメール・Slack で通知するアラートを設定

### コンテンツ管理
ポータルサイトのコンテンツ（お知らせ・ドキュメント）の管理を行います。

- **お知らせ投稿**: メンテナンス通知・新機能のお知らせを投稿する
- **ドキュメント管理**: ヘルプページやガイドのコンテンツを更新する

## アクセス権限

nagiyu Admin へのアクセスには管理者権限が必要です。管理者権限は既存の管理者から付与されます。通常の nagiyu ユーザーがアクセスしようとすると、「権限がありません」エラーが表示されます。

## セキュリティ

管理画面へのアクセスはすべて nagiyu Auth による認証を経由し、すべての操作が監査ログに記録されます。不審なアクセスを検知した場合は自動的にアラートが送信されます。

## 対象ユーザー

nagiyu Admin は、nagiyu プラットフォームの管理者・運営者を対象としています。一般ユーザーの利用は想定していません。
