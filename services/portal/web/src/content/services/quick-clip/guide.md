## Quick Clip の使い方ガイド

このページでは、Quick Clip を使って動画のクリップ切り出しやハイライト生成を行う手順を解説します。

## 事前準備

- 対応ブラウザ（Chrome・Firefox・Safari・Edge 最新版）をご用意ください。
- 処理したい動画ファイルを手元に準備してください。
- nagiyu アカウントへのログインが必要です（[auth.nagiyu.com](https://auth.nagiyu.com) から登録・ログインできます）。

---

## クリップ切り出しの手順

### ステップ 1: ログインとダッシュボードへのアクセス

[https://quick-clip.nagiyu.com](https://quick-clip.nagiyu.com) にアクセスし、nagiyu アカウントでログインします。ダッシュボードに過去の処理履歴が表示されます。

### ステップ 2: 動画のアップロード

1. 「新しいクリップを作成」ボタンをクリックします。
2. ファイル選択ダイアログから動画ファイルを選択するか、ドラッグ＆ドロップでアップロードエリアに動画をドロップします。
3. 対応フォーマット: MP4・MOV・AVI・MKV・WebM
4. 最大ファイルサイズ: 5 GB（大容量動画も対応）

### ステップ 3: クリップ区間の指定

1. アップロード完了後、動画プレビューが表示されます。
2. タイムラインを操作して「開始時間」と「終了時間」を指定します。
3. 時間は直接テキスト入力（例: `00:01:30`）でも指定できます。
4. プレビュー再生で切り出し区間を確認してから進めることを推奨します。

### ステップ 4: 出力設定

- **出力フォーマット**: MP4（推奨）・WebM・MOV から選択
- **解像度**: 元の解像度を維持、または 1080p・720p・480p に変換
- **ファイル名**: 出力ファイルの名前を指定（省略可）

### ステップ 5: 処理の開始と完了待ち

1. 「クリップを生成する」ボタンをクリックします。
2. AWS Batch でのジョブが開始され、処理状況がリアルタイムで表示されます。
3. 処理時間は動画の長さ・解像度によって異なります（目安: 10 分の動画で約 2〜5 分）。
4. 処理完了時にブラウザ通知が届きます（通知を許可している場合）。

### ステップ 6: ダウンロード

処理完了後、「ダウンロード」ボタンをクリックしてファイルを保存します。生成されたファイルは 7 日間保存されます。

## ハイライト自動生成の手順

### ステップ 1〜2: アップロードまでは同様

クリップ切り出しと同様に動画をアップロードします。

### ステップ 3: ハイライト生成モードの選択

「ハイライト自動生成」タブを選択します。

### ステップ 4: ハイライト設定

- **生成クリップ数**: 抽出するハイライトシーンの数（1〜10 個）
- **クリップ長**: 各ハイライトの長さ（10 秒〜2 分）
- **検出モード**: 音声盛り上がり検出 / 動き検出 / 両方

### ステップ 5: 処理の実行とダウンロード

「ハイライトを生成する」ボタンをクリックして処理を開始します。完了後、各ハイライトクリップを個別にダウンロードできます。

---

## 処理状況の確認

ダッシュボードの「処理履歴」タブで過去のジョブの状態を確認できます。

| ステータス | 説明                     |
| ---------- | ------------------------ |
| 待機中     | ジョブキューに登録済み   |
| 処理中     | AWS Batch で実行中       |
| 完了       | ダウンロード可能         |
| 失敗       | エラー発生（再試行可能） |
