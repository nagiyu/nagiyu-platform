## Stock Tracker とは

Stock Tracker は、指定した銘柄の株価が設定した価格帯に達したときに、ブラウザの Web Push 通知でリアルタイムにお知らせする株価追跡サービスです。常にチャートを眺めていなくても、重要な価格変動を見逃さずに把握できます。

VAPID（Voluntary Application Server Identification）キーを使ったセキュアな Web Push 通知を実装しており、対応ブラウザであればデスクトップ・スマートフォンを問わず通知を受け取ることができます。

## 主な機能

### 株価アラート設定
監視したい銘柄と価格条件を設定するだけで、株価が指定した範囲に入ったときに即座に通知します。

- **上限アラート**: 株価が設定した上限価格を上回ったときに通知
- **下限アラート**: 株価が設定した下限価格を下回ったときに通知
- **価格帯アラート**: 設定した価格帯（上限〜下限）に入ったときに通知
- **変動率アラート**: 前日比の変動率が設定値を超えたときに通知

### リアルタイム株価表示
ダッシュボードで登録した銘柄の株価をリアルタイムで一覧表示します。チャートビューと一覧ビューを切り替えて確認できます。

### Web Push 通知
ブラウザの Web Push API を活用した通知機能により、サイトを開いていない状態でも価格アラートを受け取れます。通知をクリックすると、該当銘柄の詳細ページへ直接ジャンプします。

### ウォッチリスト管理
よく見る銘柄をウォッチリストに登録して、一目で株価状況を把握できます。複数のウォッチリストを作成して用途別に管理することも可能です。

## 対応市場・銘柄

- **国内市場**: 東証プライム・スタンダード・グロース市場の全銘柄
- **米国市場**: NYSE・NASDAQ 上場銘柄
- **指数**: 日経平均・TOPIX・S&P 500・NASDAQ 100

## 技術的な特長

### AWS による高可用性アーキテクチャ
株価データは定期的に AWS Batch で取得・更新されます。バッチ処理により、市場時間中は短い間隔でデータを更新し、市場閉場後はより長い間隔に切り替えることでコストを最適化しています。

### セキュアな Web Push 通知
VAPID キーを使った認証により、プッシュ通知の送信元を検証できます。ユーザーの通知サブスクリプションは暗号化されて保存され、安全に管理されます。

## 対象ユーザー

- 個人投資家・株式投資初心者
- デイトレーダー・スウィングトレーダー
- 長期投資で特定の買い時を待っている方
- 価格アラートを活用したい投資家
