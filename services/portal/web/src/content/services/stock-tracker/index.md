## Stock Tracker とは

Stock Tracker は、指定した銘柄の株価が設定した価格帯に達したときに、ブラウザの Web Push 通知でリアルタイムにお知らせする株価追跡サービスです。常にチャートを眺めていなくても、重要な価格変動を見逃さずに把握できます。

VAPID（Voluntary Application Server Identification）キーを使ったセキュアな Web Push 通知を実装しており、対応ブラウザであればデスクトップ・スマートフォンを問わず通知を受け取ることができます。

## 主な機能

### 株価アラート設定

監視したい銘柄と価格条件を設定するだけで、株価が指定した価格に達したときに即座に通知します。

- **上限アラート**: 株価が設定した上限価格を上回ったときに通知
- **下限アラート**: 株価が設定した下限価格を下回ったときに通知
- **価格帯アラート**: 設定した価格帯に株価が入ったとき（または外れたとき）に通知
- **パーセンテージ指定**: 基準価格から何パーセント変動した価格でアラートを設定することも可能

### 株価チャート表示

取引所とティッカーを選択すると、インタラクティブな株価チャートを表示します。1 分足・5 分足・1 時間足・日足のタイムフレームに対応しています。チャートには自動更新機能があり、有効にすると 10 秒ごとにデータが再取得・再描画されます。

### Web Push 通知

ブラウザの Web Push API を活用した通知機能により、サイトを開いていない状態でも価格アラートを受け取れます。通知をクリックすると、該当銘柄のチャート画面に直接ジャンプします。

### 保有株式管理

保有銘柄・数量・平均取得価格を登録できます。登録した保有情報はチャート画面にオーバーレイ表示され、現在価格との比較に活用できます。

### 日次サマリーと AI 解析

取引時間が終了した銘柄について、日次の OHLCV（始値・高値・安値・終値・出来高）データを自動生成します。あわせてパターン分析と AI による相場解析を表示します。

## 対応市場・銘柄

対応する取引所と銘柄は管理者が登録する形式です。TradingView API が対応する取引所・ティッカーであれば登録して利用できます。

## 技術的な特長

### Lambda + EventBridge による定期処理

アラートチェックは EventBridge Scheduler により定期実行される Lambda バッチで行います。1 分間隔バッチ（`MINUTE_LEVEL`）と 1 時間間隔バッチ（`HOURLY_LEVEL`）の 2 系統があり、アラート設定時に頻度を選択できます。

### セキュアな Web Push 通知

VAPID キーを使った認証により、プッシュ通知の送信元を検証できます。ユーザーの通知サブスクリプションは暗号化されて保存され、安全に管理されます。

## 対象ユーザー

- 個人投資家・株式投資初心者
- デイトレーダー・スウィングトレーダー
- 長期投資で特定の買い時を待っている方
- 価格アラートを活用したい投資家

## 開発の背景

私がこのツールを作ったのは、株価を常に監視し続けなければならないストレスを解消したかったからです。「あの銘柄が特定の価格になったら買おう」と思っていても、日中は仕事中で画面を見られないことが多く、気づいたときにはタイミングを逃してしまうことが繰り返しありました。

そこで、価格条件を事前に設定しておけばブラウザの通知で知らせてくれるツールを自分用に作りました。既存の finance リポジトリで試作していた機能を、このプラットフォームに移行・整備したのが Stock Tracker です。

通知を受け取るだけでなく、日々の引け後に AI が相場を解析してサマリーを作ってくれる仕組みも加えました。翌朝の判断材料になるよう、直近ニュースと組み合わせた解析コメントを自動生成しています。

## 想定ユーザー

私自身がプライマリーユーザーとして使っていますが、同じ悩みを抱える個人投資家の方にも届けばと思っています。

具体的には、日中は仕事や学業で相場をリアルタイムに追えないが、「この価格になったら動きたい」という明確な価格ポイントを持っている方に向いています。また、毎日のチャートチェックを日課にしているが手動でのデータ収集が手間になっている方にも、日次サマリーと AI 解析が役立つと思います。

テクニカル分析を本格的に行う方や、証券会社のトレードツールを使いこなしている方よりは、シンプルなアラートと引け後のサマリー確認を日常に取り入れたい方を想定しています。

## 他ツールとの違い

本ツール固有の特徴として、価格アラートと AI 解析サマリーが同一のサービスで完結している点を挙げられます。一般的な証券会社のアプリでも価格アラートは提供されていますが、AI による日次解析との組み合わせや、自分でカスタマイズした銘柄リストに特化した運用は、本ツールの方が柔軟に設定できます。また、nagiyu プラットフォームの他サービスと同一の認証基盤で動作するため、追加のアカウント登録が不要です。
