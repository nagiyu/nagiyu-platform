## nagiyu Tools とは

nagiyu Tools は、日常的な開発業務やデータ処理で役立つユーティリティツールをブラウザ上で提供するサービスです。インストールや会員登録は一切不要で、ページにアクセスするだけですぐに利用を開始できます。すべての処理はブラウザ内で完結するため、入力したデータがサーバーに送信されることはなく、プライバシーの観点でも安心してご利用いただけます。

## 提供ツール一覧

### 乗り換え変換ツール

鉄道路線の乗り換え情報を整形・変換するツールです。コピーした乗り換え案内テキストを貼り付けることで、見やすい形式に自動整形します。

### JSON 整形ツール

JSON 文字列をインデント付きで整形（Pretty Print）するツールです。圧縮された JSON の可読性を高めたい場合や、API レスポンスを確認したい場面で活躍します。バリデーション機能も備えており、構文エラーをその場で確認できます。

### VAPID キー生成ツール

Web Push 通知の実装に必要な VAPID（Voluntary Application Server Identification）キーペアをブラウザ上で生成するツールです。公開鍵・秘密鍵を即座に生成し、そのままコピーして利用できます。

### Base64 エンコード／デコードツール

テキストや文字列を Base64 形式にエンコード、またはデコードするツールです。API 認証やデータ転送で Base64 形式が必要な場面で手軽に利用できます。

### URL エンコードツール

URL に含めることのできない文字（日本語・記号など）をパーセントエンコーディング形式に変換、またはその逆変換を行うツールです。クエリパラメータの作成やデバッグに役立ちます。

### ハッシュ生成ツール（SHA-256 / SHA-512）

入力したテキストの SHA-256・SHA-512 のハッシュ値（Hex 形式）を生成するツールです。データの同一性確認や検証作業の補助として活用できます。

### タイムスタンプ変換ツール

Unix タイムスタンプ（秒・ミリ秒）と日時文字列を相互変換するツールです。API レスポンスのデバッグや、ログ調査で時刻を確認したい場面で役立ちます。

## ブラウザ完結型の安心設計

nagiyu Tools の最大の特長は、すべての処理がクライアントサイド（ブラウザ）で完結する点です。入力したテキストや機密情報はネットワークを通じて送信されないため、業務上の機密データを扱う際にも安心して利用できます。また、オフライン環境でも一度ページを読み込めば動作します。

- データ処理・変換作業を頻繁に行う方

nagiyu Tools は今後もツールの種類を拡充していく予定です。ご要望やフィードバックはサポートページからお寄せください。

## 開発の背景

自分が開発業務でよく使う小さな変換・確認作業——乗り換えルートのテキスト整形、JSON の確認、Base64 やハッシュの即席確認——は、毎回ターミナルを開いたりオンラインサービスを探したりしなければならず、地味にストレスでした。「自分が使いたいものを、自分で作る」という方針で、まず自分が一番使う乗り換え変換ツールから着手し、その後もニーズを感じるたびにツールを積み重ねてきました。

すべての処理をブラウザ内で完結させているのも、機密データをサービスに渡す懸念を感じたくなかったからです。コードをサーバーに投げる前に手元で確認したい、という場面でも安心して使えるよう設計しています。

## 想定ユーザー

私自身が最初のユーザーですが、同じような悩みを持つ開発者の方にも届いてほしいと思っています。特に、ちょっとした変換・確認作業をわざわざ CLI やサービス登録なしにサクッと片付けたい方、また業務データをなるべく外部に渡したくないと考えているエンジニアの方に向いています。非開発者の方でも、Base64 の内容を確認したり乗り換えルートを整形したりといった場面で使えるよう、操作はできるだけシンプルに設計しています。
