## nagiyu Tools の使い方ガイド

このページでは、nagiyu Tools が提供する各ユーティリティツールの具体的な使い方を解説します。

## ツールへのアクセス方法

1. ブラウザで [https://tools.nagiyu.com](https://tools.nagiyu.com) にアクセスします。
2. トップページに表示されるツール一覧から、使いたいツールを選択します。
3. 各ツールページで入力フィールドにデータを貼り付けるか入力して処理を実行します。

---

## JSON 整形ツールの使い方

### 基本操作

1. 「JSON 整形」ツールを選択します。
2. 左側の入力エリアに整形したい JSON 文字列を貼り付けます。
3. 「整形する」ボタンをクリックすると、右側に整形済みの JSON が表示されます。
4. 「コピー」ボタンで結果をクリップボードにコピーできます。

### バリデーション機能

JSON の構文に誤りがある場合、エラーメッセージが表示されます。エラーの行番号と内容を確認して修正してください。

## Base64 エンコード／デコードツールの使い方

### エンコード

1. 「Base64」ツールを選択します。
2. 「エンコード」タブを選択し、テキストを入力します。
3. 「変換する」ボタンをクリックすると Base64 エンコード済みの文字列が得られます。

### デコード

1. 「デコード」タブを選択します。
2. Base64 形式の文字列を入力し、「変換する」をクリックします。
3. デコードされた元のテキストが表示されます。

---

## URL エンコードツールの使い方

### エンコード

クエリパラメータなどに含める URL エンコードが必要な文字列（日本語など）を入力し、「エンコード」ボタンをクリックします。`%E3%81%82` のようなパーセントエンコーディング形式に変換されます。

### デコード

パーセントエンコーディングされた文字列を入力し、「デコード」ボタンをクリックすると、元の文字列に戻します。

---

## ハッシュ生成ツールの使い方

1. 「ハッシュ生成」ツールを選択します。
2. ハッシュ化したいテキストを入力します。
3. アルゴリズム（MD5・SHA-1・SHA-256 など）を選択します。
4. 「生成する」ボタンをクリックするとハッシュ値が表示されます。

### 注意事項

- ハッシュは一方向性のため、ハッシュ値から元テキストを復元することはできません。
- パスワードのハッシュ化には、より強固なアルゴリズム（bcrypt など）の使用を推奨します。

---

## VAPID キー生成ツールの使い方

1. 「VAPID キー生成」ツールを選択します。
2. 「キーペアを生成する」ボタンをクリックします。
3. 公開鍵（Public Key）と秘密鍵（Private Key）がそれぞれ表示されます。
4. 各キーのコピーボタンで個別にコピーできます。

### 生成したキーの使い方

- **公開鍵**: フロントエンドの `applicationServerKey` に設定します。
- **秘密鍵**: サーバーサイドの Web Push ライブラリ（`web-push` など）に設定します。
- 秘密鍵は厳重に管理し、外部に公開しないようにしてください。

---

## 乗り換え変換ツールの使い方

1. 「乗り換え変換」ツールを選択します。
2. 乗り換え案内サービスからコピーしたテキストを入力欄に貼り付けます。
3. 「変換する」ボタンをクリックすると、整形された乗り換え情報が表示されます。

---

## よくある操作上のヒント

- **オフライン利用**: 一度ページを読み込んだ後はオフラインでも動作します。
- **データの安全性**: すべての処理はブラウザ内で完結し、サーバーへのデータ送信はありません。
- **スマートフォン対応**: スマートフォンのブラウザからも各ツールを利用できます。
