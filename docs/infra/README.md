## インフラディレクトリ構造

```
infra/
├── shared/           # 全サービスで共有するリソース
│   ├── iam/         # IAM ユーザー、ポリシー
│   └── vpc/         # VPC 関連（将来）
│
└── app-A/           # アプリケーション固有のリソース（将来）
```
