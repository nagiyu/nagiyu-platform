## カテゴリ分類

### データ変換
JSON、XML、YAML等のデータフォーマット変換ツール

### エンコーディング
Base64、URL、Unicode等のエンコード/デコードツール

### セキュリティ
ハッシュ生成、JWT、暗号化/復号化等のセキュリティ関連ツール

### ユーティリティ
UUID生成、タイムスタンプ変換等の汎用ツール

### デザイン
カラーコード変換、画像処理等のデザイン関連ツール

## 新しいツールの追加

新しいツールを追加する際は、以下の手順で進めてください：

1. **要件定義**: [requirements.md](../requirements.md) に機能要件を追加
2. **基本設計**: [basic-design.md](../basic-design.md) にAPI/画面設計を追加
3. **詳細設計**: [detailed-design.md](../detailed-design.md) に詳細仕様を追加
4. **このカタログを更新**: 実装予定または実装済みに追加
5. **実装**: [implementation.md](../implementation.md) のガイドに従って実装
6. **テスト**: [testing.md](../testing.md) に基づいてテスト
7. **カタログ更新**: 実装完了後、「実装済み」に移動
