# ツールカタログ

本ドキュメントは、Toolsアプリケーションで提供される個別ツールの一覧と概要を記録します。

実装が進むにつれて、このカタログを更新してください。

---

## ツール一覧

| ツール名 | 説明 | カテゴリ | URL |
|---------|------|---------|-----|
| 乗り換え変換ツール | 乗り換え案内のテキストを整形された形式に変換 | ユーティリティ | `/transit-converter` |
| JSON 整形ツール | JSON を整形・圧縮・検証 | データ変換 | `/json-formatter` |
| Base64 エンコーダー/デコーダー | Base64エンコード/デコード | エンコーディング | `/base64` |
| URL エンコーダー/デコーダー | URLパーセントエンコード/デコード | エンコーディング | `/url-encoder` |
| ハッシュ生成ツール | SHA-256/SHA-512 Hex 形式のハッシュ生成 | セキュリティ | `/hash-generator` |
| VAPID キー生成ツール | Web Push 通知用の公開鍵/秘密鍵ペアを生成 | セキュリティ | `/vapid-generator` |
| タイムスタンプ変換ツール | Unix タイムスタンプと日時文字列の相互変換 | ユーティリティ | `/timestamp-converter` |

---

## カテゴリ分類

### データ変換
JSON、XML、YAML等のデータフォーマット変換ツール

### エンコーディング
Base64、URL、Unicode等のエンコード/デコードツール

### セキュリティ
ハッシュ生成、JWT、暗号化/復号化等のセキュリティ関連ツール

### ユーティリティ
UUID生成、タイムスタンプ変換、乗り換え案内変換等の汎用ツール

### デザイン
カラーコード変換、画像処理等のデザイン関連ツール
