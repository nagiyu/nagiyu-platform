# Stock Tracker 一時アラート削除バッチ エラー修正 要件定義書

---

## 1. ビジネス要件

### 1.1 背景・目的

Stock Tracker の一時通知アラート失効バッチ（`temporary-alert-expiry`）が Lambda 実行時に
以下のエラーで失敗し、一時アラートが正常に無効化されていない。

```
データベースエラーが発生しました:
エンティティデータが無効です: フィールド "SubscriptionEndpoint" が文字列ではありません
```

このバッチは EventBridge Scheduler から毎時実行され、取引終了後に一時アラートを自動無効化する
重要な処理である。失敗し続けることで、一時アラートが期限切れになっても有効なまま残り、
不要な通知送信処理が発生するリスクがある。

### 1.2 対象ユーザー

- Stock Tracker システム（バッチ処理）

### 1.3 ビジネスゴール

- 一時通知アラート失効バッチが正常に完了すること
- DynamoDB に不正データが存在しても、有効なアラートは正常に処理されること

---

## 2. 機能要件

### 2.1 ユースケース

#### UC-001: 一時通知アラート失効バッチの正常実行

- **概要**: DynamoDB に不正な Web Push サブスクリプションデータが含まれていても、
  バッチ全体が失敗せず、有効なアラートを正常に処理できる
- **アクター**: EventBridge Scheduler（Lambda トリガー）
- **前提条件**: DynamoDB に `SubscriptionEndpoint` が文字列でない（または欠損している）
  アラートレコードが存在する
- **正常フロー**:
    1. バッチが起動し、`MINUTE_LEVEL` および `HOURLY_LEVEL` アラートを DynamoDB から取得する
    2. データ不正なアラートレコードはスキップされ、警告ログが出力される
    3. 有効なアラートに対して取引時間・期限チェックを行い、必要に応じて無効化する
    4. バッチが正常終了し、統計情報がログ出力される
- **代替フロー**: 有効なアラートが 0 件の場合は何もせず正常終了する
- **例外フロー**: DynamoDB への接続エラーなど、スキップ不可なエラーは従来通り
  バッチ全体のエラーとして記録される

### 2.2 機能一覧

| 機能ID | 機能名 | 説明 | 優先度 |
| ------ | ------ | ---- | ------ |
| F-001 | 不正データのスキップ処理 | `InvalidEntityDataError` が発生したアラートレコードをスキップし、バッチを継続する | 高 |
| F-002 | 不正データのログ出力 | スキップしたレコードの PK/SK とエラー内容を警告ログに出力する | 高 |

## 3. 非機能要件

### 3.1 パフォーマンス要件

| 項目 | 要件 |
| ---- | ---- |
| バッチ処理時間 | 既存の制限（Lambda タイムアウト）内に完了すること |

### 3.2 セキュリティ要件

- 不正データのスキップ処理により、既存のセキュリティ挙動は変更しない

### 3.3 可用性要件

| 項目 | 要件 |
| ---- | ---- |
| バッチ成功率 | 不正データが存在しても毎時バッチが正常完了すること |

### 3.4 保守性・拡張性要件

- `InvalidEntityDataError` の検出方法を ESM/CJS 環境に依存しない実装にする
- 不正データの原因調査を可能にするため、スキップログを詳細に出力する

---

## 4. ドメインオブジェクト

| エンティティ | 説明 |
| ----------- | ---- |
| Alert | 株価アラートエンティティ（Web Push サブスクリプション情報を含む） |
| PushSubscription | Web Push 通知のサブスクリプション情報（新形式: `subscription.endpoint/keys`、旧形式: `SubscriptionEndpoint` 等） |

---

## 5. スコープ外

- ❌ DynamoDB の不正データレコードのバックフィル（データ修正）
- ❌ 旧形式フォールバックコードの削除（別タスクとして管理）
- ❌ アラート通知バッチ（`stock-alert`）への同様の修正（別 Issue で管理）

---

## 6. 用語集

| 用語 | 定義 |
| ---- | ---- |
| 一時アラート | `Temporary: true` で設定されたアラート。取引終了後に自動無効化される |
| SubscriptionEndpoint | 旧形式の Web Push サブスクリプションエンドポイント URL（移行中） |
| InvalidEntityDataError | DynamoDB Item から Entity への変換失敗を示すエラークラス（`@nagiyu/aws`） |
