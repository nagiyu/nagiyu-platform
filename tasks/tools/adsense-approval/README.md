# Tools - Google AdSense 審査対策

## 概要

Tools サービスが Google AdSense の審査に落ちている問題を解決するための対策タスク群。

## 現在の問題

Google AdSense の審査で以下の理由により不承認となっている：

```
サイトが AdSense プログラム ポリシーを遵守していることを確認してください。

- パブリッシャーのコンテンツを含まない画面における Google が配信する広告の扱い
- 有用性の低いコンテンツ
```

### 主な原因

1. **コンテンツの量と質が不足**
   - ツールが1つしかない
   - サイト説明が簡素
   - 詳細な情報ページがない

2. **必須ページが不足**
   - プライバシーポリシー
   - 利用規約
   - サイトについて/About
   - お問い合わせ

3. **ナビゲーション構造の欠如**
   - グローバルヘッダー・フッターがない
   - サイト内の移動が困難

4. **SEO とメタデータが不十分**
   - メタデータが簡素
   - robots.txt がない
   - sitemap.xml がない

5. **コンテンツの説明不足**
   - ツールの使い方の詳細説明がない
   - FAQ やヘルプセクションがない

## 実装順序

1. **Phase 1**: 必須ページの追加
   - これがないと AdSense 審査は通らない

2. **Phase 2**: サイト構造の改善
   - ナビゲーションがないとユーザビリティが低い

3. **Phase 3**: コンテンツの拡充
   - 「有用性の低いコンテンツ」の評価を改善

4. **Phase 4, 5**: SEO 対策とメタデータ改善
   - 審査通過後も継続的に改善
